# Nexprox Proxy Application
**Created by zedon** | [Official GitHub Repository](https://github.com/ZEDON-ARCH/Nexproxy)

Nexprox is a high-performance, secure proxy and tunneling solution designed for global anonymity and network reliability.

## Features
- **Global Mesh Network**: Connect through 20+ secure global nodes.
- **Multi-Protocol Support**: SOCKS5, HTTP, HTTPS, and SOCKS4 tunneling.
- **Industrial Encryption**: All traffic is routed through encrypted handshakes.
- **Real-Time Health Audit**: Automated node scanning to ensure link stability.
- **Kill Switch Protection**: Background monitor that cuts the connection if an IP breach is detected.

## Usage
1. Start the backend: `node server.js`
2. Access the dashboard: `http://localhost:3000`
3. Log in with your credentials.
4. Select a node and protocol to initialize the tunnel.

## Privacy Policy
Nexprox, created by **zedon**, is built on a zero-logs philosophy. We do not store, monitor, or intercept user traffic. 
- **Session Data**: Temporary and stored in-memory; cleared upon server restart.
- **IP Masking**: Your real IP is never exposed to the destination server.
- **Encryption**: End-to-end handshake ensures no middle-man interception.
